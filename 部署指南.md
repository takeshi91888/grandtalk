# 🚀 亲孙通部署指南（分享给朋友使用）

## 一、准备工作（只需做一次）

### 1. 注册免费账号
- **GitHub**：https://github.com（存放代码）
- **Railway**：https://railway.app（运行 App，用 GitHub 登录即可）

### 2. 获取 Anthropic API Key
1. 打开 https://console.anthropic.com
2. 注册账号（有免费额度）
3. 点击 **API Keys** → **Create Key**
4. 复制保存好这个 key（以 `sk-ant-` 开头）

---

## 二、上传代码到 GitHub

打开终端，进入 grandtalk 文件夹，依次执行：

```bash
git init
git add .
git commit -m "亲孙通 Phase 1"
```

然后在 GitHub 新建一个仓库（名字叫 grandtalk），再执行：

```bash
git remote add origin https://github.com/你的用户名/grandtalk.git
git push -u origin main
```

---

## 三、部署到 Railway（5分钟搞定）

1. 打开 https://railway.app，用 GitHub 登录
2. 点击 **New Project** → **Deploy from GitHub repo**
3. 选择 `grandtalk` 仓库
4. Railway 会自动检测并开始构建

### 设置 API Key（关键步骤）
1. 项目部署后，点击服务 → **Variables** 标签
2. 点击 **Add Variable**
3. 填入：
   - Key：`ANTHROPIC_API_KEY`
   - Value：您的 API Key（`sk-ant-...`）
4. 点击保存，Railway 会自动重启

### 获取分享链接
部署完成后，点击 **Settings** → **Domains** → **Generate Domain**

您会得到一个类似 `grandtalk-xxx.railway.app` 的网址，把这个发给朋友就行了！

---

## 四、费用估算

- **Railway**：每月 $5 免费额度，实验性使用完全够
- **Anthropic API**：每次对话约 $0.001，100次对话约 $0.1（一毛钱）

---

## 五、本地开发（自己用）

```bash
# 确保 .env 里有 ANTHROPIC_API_KEY
npm run dev
# 打开 http://localhost:5173
```
